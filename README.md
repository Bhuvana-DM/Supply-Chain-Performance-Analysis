# Supply Chain Performance Analysis

## Overview
End-to-end data analysis project on a real supply chain dataset to uncover 
revenue trends, supplier performance, and defect rates.

## Tools Used
- **Excel** — Data cleaning and exploration
- **SQL** — Business insights and KPI extraction  
- **Power BI** — Interactive dashboard with drill-down filters

## Dataset
- 3,000+ rows of real supply chain data
- Columns: Product type, SKU, Price, Revenue, Supplier, Lead times, 
  Defect rates, Shipping carrier, Location, Stock levels

## SQL Analysis
14 queries covering:
- Revenue by product type and location
- Supplier lead times and defect rates
- Stock levels and order quantities
- Shipping carrier performance
- Inspection results

## Dashboard
Interactive Power BI dashboard with filters by product, supplier, and location.

## Key Insights
- Identified top performing product categories by revenue
- Flagged high defect rate suppliers
- Analyzed shipping delays by location and carrier
